* `clients` — хранит клиентов клуба. 
* `trainers` — хранит тренеров. 
* `services` — хранит виды тренировок: индивидуальные и групповые. 
* `membership_plans` — хранит шаблоны абонементов: цена, срок, лимит посещений. 
* `client_memberships` — хранит абонементы, которые купили конкретные клиенты. 
* `trainer_work_slots` — хранит рабочие интервалы тренеров, чтобы видеть, когда они доступны для записи. 
* `sessions` — хранит конкретные тренировки на дату и время с конкретным тренером. 
* `bookings` — хранит записи клиентов на тренировки. 
* `one_time_visits` — хранит разовые посещения зала без тренировки.
